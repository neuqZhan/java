##### 将包装类转换成其他数据类型

很简单，我们来看一个例子就知道了：

```
Integer i = new Integer(100);//转换成double类型double d = i.doubleValue();System.out.println("d的值：" + d);//转换成float类型float f = i.floatValue();System.out.println("f的值" + f);
```

相信你已经明白怎么进行转换了。
